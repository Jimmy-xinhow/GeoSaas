- [ ] 用戶可通過 Google OAuth 登入
- [ ] 登入後可看到 Dashboard
- [ ] 用戶可新增/刪除監控的網站
- [ ] 爬蟲可成功抓取並分析網頁的 8 項指標
- [ ] 掃描結果可正確儲存到資料庫
- [ ] Landing Page 可正常訪問

- [ ] CI Pipeline 正常運行（Lint + Test + Build）
- [ ] Staging 環境可正常訪問
- [ ] Docker Compose 可一鍵啟動本地開發環境

- [ ] 單元測試覆蓋率 > 60%（核心模組）
- [ ] 無 P0 級 Bug
