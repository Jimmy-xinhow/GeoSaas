# GEO SaaS - 團隊 KPI 與績效指標

---

## 一、團隊整體 OKR（季度）

### Q1 目標：MVP 上線 + 首批付費用戶

| Objective | Key Result | 衡量方式 |
|-----------|-----------|---------|
| 產品 MVP 按時上線 | Alpha 版在第 8 週上線 | 上線日期 |
| 用戶增長 | 內測期間獲得 100+ 註冊用戶 | 用戶數據 |
| 產品品質 | 上線後 P0 Bug 數 = 0 | Bug Tracker |
| 用戶滿意度 | NPS ≥ 30 | 用戶調查 |

### Q2 目標：正式版上線 + 商業驗證

| Objective | Key Result | 衡量方式 |
|-----------|-----------|---------|
| 正式版上線 | v1.0 在第 16 週上線 | 上線日期 |
| 營收驗證 | 月營收 (MRR) 達 $5,000 | Stripe Dashboard |
| 用戶留存 | 月留存率 ≥ 60% | Analytics |
| 功能完整 | 內容引擎 + 監控系統上線 | 功能驗收 |

## 四、績效評估週期

| 評估類型 | 頻率 | 參與者 | 內容 |
|---------|------|--------|------|
| Sprint Review | 每 2 週 | 全員 | 交付物驗收、KPI 數據 |
| 1-on-1 面談 | 每月 | PM + 個人 | 工作狀態、成長需求 |
| 團隊回顧 | 每月 | 全員 | 協作效率、流程改善 |
| OKR Review | 每季 | 管理層 + 全員 | 目標達成度、下季規劃 |
